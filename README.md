# ?? Schrute Farms — The Fullstack Game

> "Beterraba. É o que mais cresce. É o que mais importa." — Dwight K. Schrute

## Visão Geral
Gerencie a Schrute Farms: plante beterrabas, regue, colha e acumule pontos.
Um jogo fullstack com motor baseado em Canvas, API REST e autenticação de fazendeiro.

## Tecnologias
| Camada    | Stack                          |
|-----------|-------------------------------|
| Frontend  | Vite · React · TypeScript      |
| Backend   | Node · Express · TypeScript    |
| Banco     | In-memory (JSON file persist)  |

## Início rápido
```bash
npm install
npm run dev
```

## Estrutura
```
packages/
  backend/   ? API Express
  frontend/  ? Jogo React + Canvas
```
